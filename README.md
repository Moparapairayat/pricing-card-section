## ✨ Next-Gen Feature Matrix

| Icon | Feature | Technical Highlights & Experience |
| :---: | :--- | :--- |
| ⚡ | **1500ms Count-Up Animation** | Eased `requestAnimationFrame` tween counts up from `0` to target price on page load and smoothly transitions over `800ms` on currency/billing changes. |
| 🔄 | **Smart Billing Toggle** | Interactive spring switch toggling Monthly vs. Yearly pricing with per-tier **Save 20%** badges and recalculated daily-equivalent breakdowns (`≈ ৳14/day`). |
| 🌐 | **Dynamic Currency Switcher** | Custom glassmorphic dropdown support for **BDT (৳), USD ($), EUR (€), TRY (₺), IRR (﷼)** with country-accurate formatting & decimal precision. |
| 🧲 | **Magnetic CTA Button** | Flagship CTA button physically tracks cursor proximity (capped at `±15px`) using friction math for a ultra-tactile feel. |
| 🔮 | **Electric Displacement Border** | SVG filter `feTurbulence` + `feDisplacementMap` creates a live, glowing electric border for the flagship plan. |
| 🌟 | **Cursor Spotlight Glow** | Radial mouse-tracking light follows the cursor across the cards via CSS `--mouse-x` & `--mouse-y` custom properties. |
| 👻 | **Peer Card Dimming** | Hovering over a card gently dims (`opacity: 0.6`) and desaturates non-focused cards using modern CSS `:has()` selector. |
| 💡 **Hover Tooltips** | CSS-driven, zero-JS glassmorphic tooltips offering deep context on product capabilities. |
| 💚 | **Savings Tracker Banner** | Coupon-style pulsing banner calculating exact yearly savings with green glow animation. |
| 🌌 | **Ambient Background Layer** | 3-layer background system consisting of drifting nebula blobs, grid overlay, mouse glow, and noise texture. |

## ⚡ Quick Integration Guide

### 1. HTML5 (Zero-Build Vanilla)
Simply open `html/pricing2.html` in any modern web browser or serve it statically.

### 2. React 18
Copy `react/PricingSection.jsx` & `react/PricingSection.css` to your components directory:
```jsx
import PricingSection from './components/PricingSection';

export default function App() {
  return <PricingSection />;
}
```

### 3. Vue 3 (Composition API)
Copy `vue/PricingSection.vue` & `vue/PricingSection.css`:
```vue
<script setup>
import PricingSection from './PricingSection.vue';
</script>

<template>
  <PricingSection />
</template>
```

### 4. Next.js 14 (App Router & TypeScript)
Copy `nextjs/components/PricingSection.tsx` & `nextjs/components/PricingSection.css`:
```tsx
import PricingSection from '@/components/PricingSection';

export default function PricingPage() {
  return <PricingSection />;
}
```

### 5. Svelte 4 / 5 (TypeScript)
Copy `svelte/PricingSection.svelte` & `svelte/PricingSection.css`:
```svelte
<script>
  import PricingSection from '$lib/components/PricingSection.svelte';
</script>

<PricingSection />
```

### 6. Astro
Copy `astro/PricingSection.astro` & `astro/PricingSection.css`:
```astro
---
import PricingSection from '../components/PricingSection.astro';
---

<PricingSection />
```

## 🎨 Design Tokens & Customization

The core design system is powered by CSS Custom Properties defined in `PricingSection.css`:

```css
:root {
    /* Base Color Tokens */
    --bg-base: #030303;
    --bg-surface: #0a0a0a;
    --bg-surface-elevated: #111111;

    /* Typography */
    --text-primary: #f8fafc;
    --text-secondary: #cbd5e1;
    --text-tertiary: #94a3b8;

    /* Tier Accents */
    --essential-accent: #e2e8f0;
    --elite-accent: #60a5fa;
    --ult-accent-1: #818cf8; /* Indigo */
    --ult-accent-2: #38bdf8; /* Cyan */
    --ult-accent-3: #c084fc; /* Purple */

    /* Easing Functions */
    --ease-spring: cubic-bezier(0.175, 0.885, 0.32, 1.15);
    --ease-smooth: cubic-bezier(0.25, 1, 0.5, 1);
}
```
